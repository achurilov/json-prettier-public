Welcome to JSON Prettier

Your Chrome extension for quick JSON formatting, viewing and editing.

JSON Prettier is activated automatically when you open any JSON file directly in the browser. The toolbar icon is used for extra tools and settings.

Pin JSON Prettier and get started in 3 easy steps

1

Open the extensions menu

Click the puzzle icon in the top-right corner of Chrome to open the list of installed extensions.

2

Pin JSON Prettier to your toolbar

In the extensions list, find JSON Prettier and click the pin icon to keep it visible on your toolbar.

3

Open the extension from any page

Click the JSON Prettier icon on the toolbar to open the extension on any page and create a new JSON file to edit.

How the toolbar icon works

  • Automatic formatting: when you open a JSON file in Chrome, JSON Prettier formats and displays it automatically.
  • Right-click the icon: open the extension configuration options.
  • Left-click the icon: create a new JSON file and edit it in the built-in editor.

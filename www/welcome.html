Follow these simple steps to pin the extension for quick access.

Pin JSON Prettier to your toolbar

  • 1 Click the puzzle icon in the top-right corner of Chrome.
  • 2 Find JSON Prettier in the list of extensions.
  • 3 Click the pin icon to keep it visible on your toolbar.
Automatic activation: JSON Prettier activates automatically whenever you open a JSON file in your browser. Use the toolbar icon for additional actions:
  1. Right-click the icon to open the extension settings.
  2. Click the icon to open a new JSON editor tab and create or edit a JSON file.

JSON Prettier is ideal for API responses, logs, configuration files, and any other JSON you need to inspect.
